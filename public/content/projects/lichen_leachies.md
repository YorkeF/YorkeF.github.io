# Lichen Leachies
## E-commerce site for reptiles and plants
**Year:** March 2026
**Status:** Active
**Source:** github.com/YorkeF/lizards

---

## Overview

Developed a full e-commerce site to assist in the sale of Leachianus
geckos, isopods, and plants. Handles product listings, inventory,
and order management.

## Features

- Product catalogue with category filtering (geckos, isopods, plants)
- Shopping cart and checkout flow
- Admin dashboard for inventory and order management
- Automated deployment via GitHub Actions

`PHP` `MySQL` `React` `GitHub Actions`
